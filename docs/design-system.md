## Quick Reference

### Key Colors

- **Brand:** `#093CB5` / **Accent Cyan:** `#00D2D8` / **Accent Orange:** `#E57300`
- **Headings:** `text-gray-500` (`#28314B`) / **Body:** `text-gray-400` (`#474F66`)
- **Borders:** `border-gray-200` (`#CFD5E4`)

### Typography

- **H1:** 48px medium / **H2:** 32px medium / **H3:** 24px semibold / **H4:** 20px semibold
- **Body:** 16px/22px / **Small:** 14px / **Large:** 20px/26px
- **Font:** Public Sans

### Card Rules

- On colored backgrounds (surface-secondary, tertiary): **no border**
- On white backgrounds: `border border-gray-200`
- Hover shadow: `0 4px 12px rgba(0,0,0,0.04)`

### Section Spacing

- Between sections: `mt-16`
- Container: `mx-auto max-w-6xl px-4 sm:px-6 lg:px-8`
